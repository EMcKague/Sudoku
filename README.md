# Sudoku
Created Sudoku in python using pygame 
## Installation
Use the package manager pip to install pygame.
```bash
pip install pygame
```
## Usage
requires python 3.9 
Run python3 Sudoku.py from command line

```
python3 Sudoku.py
```
